Software Developement Engineer(L1)

Bengaluru / Software Engineering / Fulltime
Apply Now

Roles and Responsibilities

  • The Software Development Engineer's core responsibilities include working on highly maintainable and unit-tested software components/systems that address real-world problems.
  • Experienced in microservices-based architecture, can work on cloud infrastructures like GCP, AWS etc
  • Should take the e2e ownership of product/feature right from design, code and deployment.
  • Ensure quality at every level be it problem-solving, design, code or bug fixes.
  • Should be able to collaborate with product managers, architects and other stakeholders to ensure smooth execution of sprints.
  • Own and unblock users on production issues, able to troubleshoot and fix production issues on priority.
  • Can mentor and help other team members and ensure that the overall productivity of the team is high.
  • Should be able to work independently and work on platform automation and development like building CI&CD, K8S operators, service mesh etc
  • Creating automation to help in onboarding new services with the production readiness.
  • Generating automated reports on services SLO/Error Budgets/Alerts and Operational Overhead.
  • Working with the various team to have an in-depth understanding of the application architecture and solve its bottlenecks.
  • Managing Outages, troubleshooting and doing detailed RCA with various teams to identify ways to avoid that situation.
  • Demonstrate good learnability and adopt technologies that help build large-scale, performant, reliable and sustainable systems.

Roles and Responsibilities

  • 5+ Years of experience as SDE in building scalable restful/soap microservice for large-scale infrastructure.
  • Strong fundamentals in Data Structures and Algorithms Good understanding of their space & time complexities.
  • Should have strong experience and ability to write good code with languages, such as Python, Java, Golang etc.
  • Excellent coding skills - should be able to convert the design into code fluently with strong technical aptitude and a good knowledge of CS fundamentals.
  • A collaborative spirit with the ability to work across disciplines to influence, learn and deliver.
  • A deep understanding of computer science, software development, and networking principles.
  • Good understanding of DNS, TCP/IP, UDP, GRPC, Routing and Load Balancing.
  • Expertise in Google Cloud (GCP) and/or other relevant Cloud Infrastructure solutions like AWS or Azure.
  • Experience in building CI/CD solutions with tools such as Jenkins, GitLab, Spinnaker, Argo etc.
  • Experience in managing and deploying containerized environments using Docker, and Kubernetes is a must.
  • Experience with multiple datastores is a plus (MySQL, Scylla, Cassandra, Redis, Elasticsearch).
